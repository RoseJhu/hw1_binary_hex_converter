本專案實作之進位轉換器，核心演算法完全採用自主開發之數學邏輯，不調用 Python 內建之進位轉換函式。針對「十進位轉他進位」採用輾轉相除法，「他進位轉十進位」則應用加權展開計算法。介面部分參考復古數位風格進行優化，除了滿足 0-255 的基礎需求，亦經測試確認可精確處理如 300 以上之大數值，展現了手寫演算法的穩定性與擴充性。
