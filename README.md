**一、開發說明**
：本專案實作之轉換系統，核心演算法完全採用自主開發之數學邏輯。針對「十進位轉他進位」採用輾轉相除法，「他進位轉十進位」則應用加權展開計算法。

**二、功能特色**

介面設計：參考復古數位風格調色盤，提升視覺辨識度。

大數支援：經測試可穩定處理超過 255 以上之數值（如測試範例 300）。

穩定性：具備非法字元輸入偵測與防呆機制。
